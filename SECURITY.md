# Security Policy

## 支持的版本

| 版本 | 支持状态 |
|------|---------|
| 0.0.5 | ✅ 当前版本 |
| < 0.0.5 | ❌ 不再维护 |

## 报告安全漏洞

如果你发现了安全漏洞，**请不要通过公开 Issue 报告**。

请发送邮件至 **[daixueyun3377@gmail.com](mailto:daixueyun3377@gmail.com)**，包含：

- 漏洞描述
- 复现步骤
- 影响范围
- 可能的修复方案（如有）

我们会在 **48 小时内** 确认收到报告，并在 **7 个工作日内** 给出初步评估。

## 安全最佳实践

使用 AgentMark 时请注意：

- **不要在代码中硬编码 API Key**，使用环境变量或配置中心
- **限制 @AgentMark 工具的权限范围**，避免暴露敏感操作
- **在生产环境中对 Agent API 接口做鉴权**，防止未授权调用
